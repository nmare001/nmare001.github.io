I'm a computer programmer from Southern California. This is my portfolio of projects, music, and sound effects that I've worked on over the last few years.

Projects

These are some of the projects I've worked on both as a student at UCR and outside of school.

Echo in the Mirror: A top-down RPG in the style of Link to the Past and Chrono Trigger. I worked on music and sound effects for this game.

Drum Machine Project: A drum machine built using arduino parts and programmed in C using Atmel Studio. User can program a unique sequence of drum sounds which can be sped up or slowed down.

VR Trivia Game: A VR game for the Oculus Rift built in Unity. The player answers trivia questions to get from one themed room to the next eventually finishing the game.

CS152 Compiler Project: Group project where we built a compiler in C++ using Flex/Bison. It converts commands into machine code.

Remote Shell Program: Group programming project where we built a program in C++ which creates a command shell called rshell. This program handles piping, input redirection, and output redirection.

Sound Effects

These are some some sound effects I've created. Most of them come from my work on Echo in the Mirror and Gamespawn's 2018 Fall Game Jam. I get a lot of samples from freesound.org and BATTERY drum kits. I layer these on top of eachother adding effects (mainly reverb, distortion, filters, and half-time) to make unique sounds.
